## Wkład do repozytorium

Dziękujemy za chęć współpracy! Kilka zasad, aby utrzymać porządek:

- Zgłaszaj nowe informacje przez issue lub pull request.
- Użyj odpowiedniego katalogu: `conferences/`, `trainings/`, `webinars/`, `news/`, `links/`, `files/`.
- Skopiuj i uzupełnij odpowiedni szablon z `templates/`.
- Nazewnictwo plików: `YYYY-MM-DD-krótki-tytuł.md` lub czytelny slug.
- Dołącz źródła/odnośniki i licencję materiałów, jeśli dotyczy.

Licencja treści:
- Proponuję użyć licencji treści Creative Commons (np. CC BY-SA 4.0). Jeśli wolisz inną, napisz.

Moderacja:
- Wszystkie PR będą przeglądane przed scaleniem. W razie potrzeby zaproponujemy poprawki.

Kontakt:
- Jeśli masz pytania, otwórz issue albo zostaw komentarz w PR.

Dziękujemy za wkład!
