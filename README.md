## W krainie genetyki

Repozytorium zbierające informacje o konferencjach, szkoleniach, webinarach, nowinkach i przydatnych odnośnikach związanych z genetyką w Polsce.

Struktura:
- `conferences/` — zapowiedzi i podsumowania konferencji
- `trainings/` — kursy i szkolenia
- `webinars/` — webinary i nagrania
- `news/` — krótkie nowinki i aktualności
- `links/` — przydatne strony i zasoby
- `files/` — udostępnione materiały (prezentacje, PDFy)
- `templates/` — szablony wpisów (używaj do nowych plików)

Jak CONTRIBUTING:
- Otwórz issue lub pull request z nowym wpisem.
- Użyj szablonu z `templates/` i zapisz plik w odpowiednim katalogu.
- W razie wątpliwości napisz opis w issue.
