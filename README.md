# 🧬 w_krainie_genetyki

> Jedno miejsce na polską genetykę — aktualności, konferencje, szkolenia, webinary i wydarzenia.

Strona przeznaczona dla wszystkich pracujących w genetyce (głównie człowieka). Stworzona przez diagnostę-genetyka dla genetyków. Hostowana na GitHub Pages.

**→ [w_krainie_genetyki.github.io](https://w-krainie-genetyki.github.io)**

---

## Sekcje

| Sekcja | Opis |

| **Konferencje** | Konferencje i spotkania |
| **Kursy** | Szkolenia, webinary, kursy online i stacjonarne |

## Wdrożenie na GitHub Pages

1. Utwórz repozytorium o nazwie `w-krainie-genetyki` (lub `TWOJ_USERNAME.github.io`)
2. Wgraj pliki do brancha `main`
3. W ustawieniach repozytorium: **Settings → Pages → Source: Deploy from branch → main / (root)**
4. Strona będzie dostępna pod adresem `https://TWOJ_USERNAME.github.io/w-krainie-genetyki`

> Uwaga: GitHub Pages jest już włączone dla repozytorium, więc każda zmiana wypchnięta do `main` zbuduje i opublikuje stronę automatycznie.
